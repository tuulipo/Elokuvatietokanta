# Elokuvatietokanta
Yksinkertainen tietokantasovellus, jolla käyttäjä voi pitää kirjaa häntä kiinnostavista ja hänen arvostelemistaan elokuvista.

- Sovelluksessa käyttäjät pystyvät lisäämään ja muokkaamaan elokuvia. Elokuvalle tulee antaa ainakin perustiedot (nimi, julkaisuvuosi, ohjaaja, kesto).   Elokuvalle voidaan antaa muita tietoja (mm. synopsis, pääosan esittäjä). 
- Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
- Käyttäjä pystyy selaamaan ja etsimään hakusanalla elokuvia.
- Käyttäjä pystyy arvostelemaan elokuvia (arvostelu = arvosana + halutessaan myös kommentti). Elokuvasta näytetään kommentit ja keskimääräinen arvosana.
- Käyttäjä pystyy lisäämään elokuvan suosikiksi.
- Käyttäjäsivu näyttää kuinka monta elokuvaa käyttäjä on arvostellut ja listan kyseisistä arvosteluista.
- Käyttäjä pystyy valitsemaan elokuvalle yhden tai useamman luokittelun (esim. genre, Oscar-voittaja).
- Käyttäjä pystyy tekemään muokkauspyynnön elokuvalle (esim. tietojen korjaus tai lisäys).

Pääasiallinen tietokohde on elokuva ja toissijainen tietokohde on elokuvan arvostelu.
